# Isolated Flyback Converter

## Overview

This repository contains the complete design of an isolated flyback DC-DC converter developed using KiCad. The project includes the schematic, PCB layout, manufacturing files, and supporting documentation required for fabrication and assembly.

The converter provides galvanic isolation between the input and output stages, making it suitable for embedded systems, industrial electronics, sensor interfaces, and low-power isolated supply applications.

---

## Features

- Isolated DC-DC power conversion
- Flyback converter topology
- Compact PCB design
- Designed using KiCad
- Manufacturing-ready Gerber files
